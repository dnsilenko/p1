## Нотатка щодо trigram
На поточному етапі trigram **слід тренувати** на текстах об'ємом **600-650 слів**. Це займе певний час, 
але має успішно завершитись. Було перевірено текст на 800 слів, модель тренувалась доволі довго, але впоралась. 
Текст більшого розміру в більшості випадків призведе до помилки Insufficient memory to continue the execution of the program. 

Це пов'язано зі способом організації записів probabilities. В наступній версії недолік буде покращено. Дякуємо за розуміння і за те, що продовжуєте використовувати наші мовні моделі😊.
